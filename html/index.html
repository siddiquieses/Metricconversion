THE SI SYSTEM

METRIC SYSTEM

The metric system is a decimal-based system of measurement that is widely used across the globe. It is based on powers of 10, which makes it easier to convert between different units. The system was introduced in France in the late 18th century, and it became the standard for scientific measurements worldwide.

One of the key features of the metric system is its simplicity. For example, in the metric system:

1 kilometer (km) = 1,000 meters (m)

1 liter (L) = 1,000 milliliters (mL)

1 kilogram (kg) = 1,000 grams (g)

The metric system forms the backbone of the SI system, which is the standard measurement system used in science, industry, and commerce.

THE METER

The meter is the base unit of length in the metric system and the SI system. Historically, the meter was defined in the late 18th century based on a fraction of the Earth's meridian, the distance from the North Pole to the equator. Later, the definition was refined using the Earth's dimensions.

Today, the meter is defined as the distance light travels in a vacuum in 1/299,792,458 seconds. This definition uses the constant speed of light to create an incredibly precise and reproducible unit of measurement. The meter is used worldwide in scientific, engineering, and everyday contexts.
